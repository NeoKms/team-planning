# AI memory

Эта директория хранит устойчивый контекст проекта для AI-ассистента.

Перед началом задачи агент должен прочитать:

1. `project-context.md`
2. `project-rules.md`
3. `implementation-plan.md`, если задача связана с реализацией продукта

Дополнительно:

- `decisions.md` — долгосрочные решения и причины.
- `implementation-plan.md` — живая очередь шагов реализации.
- `change-log.md` — важные изменения проекта, которые помогут будущей работе.

## Как обновлять память

- Добавляй только информацию, которая пригодится в будущих задачах.
- Формулируй коротко и проверяемо.
- Не сохраняй секреты, токены, пароли, приватные ключи и временные данные.
- Если пользователь явно просит что-то запомнить, обязательно обнови подходящий файл.
- Если изменение проекта влияет на будущую разработку, обнови память без отдельной просьбы.
